# AI Resume Screening System

A self-contained AI-style resume screening dashboard built with HTML, CSS, and JavaScript. Recruiters can upload resume text files, paste a job description, rank candidates, inspect skill matches, and view live analytics.

## Features

- Resume upload and parsing from `.pdf`, `.txt`, `.md`, `.csv`, and similar text-based files
- Job description upload from PDF or text-based files
- Job description skill extraction
- Candidate ranking by skill overlap and keyword similarity
- Interactive recruiter dashboard with candidate cards and detailed match summaries
- Real-time analytics for average score, top candidate, resume volume, and skill coverage
- Match score chart and missing-skill insights

## How To Run

Open `index.html` in a browser.

No backend or package installation is required.

## Suggested Demo Flow

1. Open `index.html`.
2. Paste a job description or upload a JD file.
3. Upload one or more resume PDF/text files, or use the sample resumes.
4. Click **Screen Resumes**.
5. Review ranked candidates, matched skills, missing skills, and analytics.
